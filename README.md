# Task Tracker (Full-Stack)

## Overview

A full-stack task management application built with:

* ASP.NET Core (.NET 10)
* PostgreSQL
* Entity Framework Core
* React (Vite)

## Features

* Create tasks
* View all tasks
* Update task completion
* Delete tasks

## Tech Stack

### Backend

* ASP.NET Core Web API
* Entity Framework Core
* PostgreSQL

### Frontend

* React
* Axios
* Vite

## How to Run

### Backend

cd TaskApi
dotnet run

### Frontend

cd task-ui
npm install
npm run dev

## API Endpoints

* GET /api/task
* GET /api/task/{id}
* POST /api/task
* PUT /api/task/{id}
* DELETE /api/task/{id}
